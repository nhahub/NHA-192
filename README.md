# DragoBlaze - Android Movie Streaming Application 

# Overview
DragoBlaze is an Android application built with Kotlin and Jetpack Compose that allows users to browse, search, and view details of movies using a real-time movie API.
The app demonstrates clean architecture using MVVM, Retrofit, Coroutines, and StateFlow for smooth asynchronous UI updates.

# Features
Search for movies across all genres.
View movie details (title, description, rating, poster).
Jetpack Compose UI with dynamic state handling.
Fetch data from API using Retrofit.
MVVM architecture with Repository pattern.
Clean and reactive UI using StateFlow.
Error & loading states.
Responsive layout for all screen sizes.

# Tech Stack
Kotlin
Jetpack Compose
MVVM architecture
Retrofit + Gson
Coroutines & StateFlow
Navigation Compose
Coil for image loading
